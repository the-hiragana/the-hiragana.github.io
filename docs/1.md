# 1 Dependencies

# 1.1 webpack
```sh
$ yarn add -D webpack webpack-dev-server
```

# 1.2 babel
```sh
$ yarn add -D babel-core babel-loader
```

```sh
$ yarn add -D babel-preset-env
```

.babelrc
```
{
  "presets": ["env"]
}
```

# 1.3 vue
```sh
$ yarn add vue
```

# 1.4 vue-loader
```sh
$ yarn add -D vue-loader css-loader vue-template-compiler
```
